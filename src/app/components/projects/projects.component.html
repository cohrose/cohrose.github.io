UNF MyWings Portal
The vendor for UNF's student portal updated their custom developemnt process, so we decided to take this opportunity to do a re-design, which meant we had to remake all our custom pages in React.
I was responsible for all the React custom pages. Since this is a vendor project, we had to adhere to their Application Extension Kit (AEK) to create most features. It was really fun switching it up to React since I was used to Angular.
